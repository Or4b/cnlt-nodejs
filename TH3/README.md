# Thực Hành 3: Xây dựng Web App bằng Express + EJS

## Thông tin sinh viên
- **Họ và tên:** Huỳnh Nguyên Gia Bảo
- **Chuyên ngành:** Hệ thống Thông tin - Đại học Quy Nhơn

## Mô tả dự án
Dự án xây dựng một Blog cá nhân cơ bản sử dụng Node.js, Express và EJS template engine. Giao diện được tích hợp từ template Clean Blog (Start Bootstrap).

## Yêu cầu đã hoàn thành
- Sử dụng layout dùng chung (header, navbar, footer).
- Render giao diện động với dữ liệu từ mảng.
- Tích hợp vòng lặp và câu lệnh điều kiện `if/else` để hiển thị trạng thái.
- Sử dụng route động `/detail/:id` để xem chi tiết bài viết.

## Hướng dẫn chạy dự án
1. Di chuyển vào thư mục TH3: `cd TH3`
2. Cài đặt các thư viện cần thiết: `npm install`
3. Khởi chạy server: `node app.js`
4. Mở trình duyệt và truy cập: `http://localhost:3000`
